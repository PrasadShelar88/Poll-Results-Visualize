# Poll Results Visualizer Frontend

A modern React + Vite frontend dashboard that matches the **Poll Results Visualizer backend**. It reads the backend's cleaned CSV structure and visualizes poll insights with an industry-style interface.

## Features
- Professional dashboard UI
- Works with the backend's `cleaned_poll_data.csv`
- Filters for Region, Age Group, and Preferred Tool
- Bar chart, pie chart, trend line, demographic analysis, and keyword insights
- Dataset preview for screenshots and demos
- Upload your own cleaned CSV file directly from the browser

## Tech Stack
- React
- Vite
- Recharts
- PapaParse
- CSS

## How this matches the backend
This frontend is designed for the backend zip you already generated. It uses the same cleaned CSV columns:
- `Timestamp`
- `Date`
- `Region`
- `Age Group`
- `Preferred Tool`
- `Satisfaction (1-5)`
- `Feedback`

You can either:
1. Use the included sample data copied from the backend outputs.
2. Upload a new `cleaned_poll_data.csv` generated by the backend.

## Recommended demo flow
1. Run backend pipeline to generate fresh outputs.
2. Run this frontend.
3. Upload the backend `cleaned_poll_data.csv` from the browser.
4. Show filters, charts, and dataset preview.
